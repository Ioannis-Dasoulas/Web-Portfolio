# Website Instructions

## Overview

This is a personal website for showcasing publications, projects, and other professional information.

## Design

- **Framework**: HTML Bootstrap 5, CSS, JavaScript
- **Theme**: Simple, clean, and professional
- **One-page website** with the following sections:
    - **Home**: Introduction and brief overview of the website
    - **About**: More detailed information about the website
    - **Publications**: List of publications with links to the full text
    - **Projects**: List of projects with links to the full text
    - **Contact/Footbar**: A footer section with contact information and social links
- **Interactive Tabs**: The "Publications" and "Projects" sections must function as independent tabs. When one is active, the items from the other category must be hidden.
- **Card Content**: Every entry in the Publications and Projects sections must include an author list, a title, a representative image, and a description.
- **Data Preservation Rule**: NO projects or publications must ever be removed, hidden, or deleted during edits. Furthermore, their exact details (titles, descriptions, links, authors) must NEVER be altered or shortened without explicit permission.
- **Responsiveness**: The website must follow mobile-first design principles. All sections, text, and elements must look excellent and be fully usable on different screen sizes, especially smartphones.
